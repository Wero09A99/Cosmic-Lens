📋 INSTRUCCIONES RÁPIDAS (3 PASOS)
═══════════════════════════════════════════════════════════════

1️⃣  INSTALAR (solo la primera vez)
   ➜ Doble clic en: INSTALAR.bat
   ➜ Espera a que termine (puede tardar 1-2 minutos)

2️⃣  INICIAR EL PROGRAMA
   ➜ Doble clic en: INICIAR.bat
   ➜ Se abrirá automáticamente en tu navegador

3️⃣  CARGAR IMÁGENES
   ➜ Opción A: Haz clic en "🌌 Datasets" (imágenes de Hubble/JWST)
   ➜ Opción B: Haz clic en "📤 Subir" para subir tus propias imágenes
   ➜ Opción C: Copia tus imágenes a la carpeta "data"

═══════════════════════════════════════════════════════════════
❓ PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════

  ❌ "Python no está instalado"
     → Instala Python desde python.org
     → Asegúrate de marcar "Add Python to PATH"

  ❌ "No se puede conectar"
     → Verifica que INICIAR.bat esté ejecutándose
     → Abre manualmente: http://localhost:5000

  ❌ "La imagen no carga"
     → Verifica que el archivo esté en la carpeta "data"
     → Formatos soportados: JPG, PNG, TIFF, BMP
